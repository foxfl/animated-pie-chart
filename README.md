# Animated Pie Chart 

Animated Pie Chart made with React Native Skia, React Native Reanimated and React Native Gesture Hanlder 



## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

3. Open in the iOS simulator by pressing `I` or scan the QR code to open on your phone
